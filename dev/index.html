JuMPIn.jl

JuMP Incidence Graph Analysis. Tools for constructing and analyzing the incidence graph or matrix of variables and constraints in a JuMP model.

If you are wondering what JuMP is, please see here. If you are familiar with JuMP and wondering whether you should use JuMPIn, you may find the Overview helpful.
